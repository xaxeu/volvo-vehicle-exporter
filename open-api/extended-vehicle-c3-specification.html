Extended Vehicle (1.0)

Download OpenAPI specification:

This API allows to get latest information of the vehicle such as distance driven, lock status, windows and doors, rooftop or sunroof and engine status of connected Volvo Cars vehicles remotely.

Vehicle-List

Get Vehicle List

Provides all current valid relations between an Volvo Id (user) and its connected vehicles. Returns a list of VIN numbers together with timestamp of relation
